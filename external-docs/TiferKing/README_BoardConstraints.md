# AS02MC04_hack

AS02MC04 board reverse engineering documentation

## Contents

- constraints: Constraints file describing pin functions.
- documents: Reference documents.
- examples: Example projects.
- as02mc04: Board files.

## Usage

Copy the as02mc04 folder to Vivado/\<version\>/data/boards/board_files/ . If the board_files directory doesn't exist, create one.

Open Vivado and create a new project. Select "AS02MC04 NIC Card."
